ANCAT System

ANCAT is a web-based annotation tool designed to identify communication gaps in job postings.

The system enables participants to highlight text spans and categorize them into multiple communication gap types.

How to Use: Clone the repository, run npm install, then npm run dev.

View ANCAT on GitHub

Dataset

This project includes four datasets capturing communication gaps in workplace contexts.

Annotation Dataset: 683 annotations across 200 job descriptions.

Interview Dataset: 720 qualitative quotes.

How to Use: Analyze communication patterns or train AI models.
